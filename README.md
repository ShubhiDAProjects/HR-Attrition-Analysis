# 👩‍💼 HR Attrition Analysis | Power BI Project

## 📌 Project Overview

This project focuses on analyzing employee attrition data to understand the factors contributing to employee turnover. An interactive Power BI dashboard is developed to visualize key HR metrics and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall attrition rate
* Identify key factors influencing employee turnover
* Evaluate attrition by department, role, and demographics
* Understand the impact of salary, experience, and job satisfaction
* Provide insights to improve employee retention

---

## 🛠️ Tools & Technologies Used

* **Power BI** → Data visualization & dashboard development

---

## 🔄 Project Workflow

### 1. Data Preparation

* Imported dataset into Power BI
* Cleaned and transformed data using Power Query
* Handled missing values and ensured data consistency

---

### 2. Data Modeling

* Created relationships between tables (if applicable)
* Designed an optimized data model

---

### 3. Dashboard Development

* KPI Cards (Attrition Rate, Total Employees, Avg Salary, Avg Tenure)
* Bar charts for department-wise and role-wise attrition
* Pie/Donut charts for gender and education distribution
* Line charts for attrition trends (if time data available)
* Slicers for filtering (department, job role, age group)

---

## 📊 Key Insights

* Identified departments with highest attrition
* Observed higher attrition in certain job roles
* Found correlation between low job satisfaction and higher attrition
* Salary and experience play a role in employee retention

---

* Developed an interactive dashboard for HR analytics
* Enabled better workforce planning and retention strategies
* Demonstrated strong Power BI and data storytelling skills
